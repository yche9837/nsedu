# NS EDU - V1 Website (<span style="background-color:#f0f0f0;">index.html</span>)

The folder contains the complete v1 static website for NS EDU. The main file is `index.html`, which includes all the HTML, CSS, and JavaScript needed for the site to function.

The entire website (Home, About Us, Classes, Resources, Contact Us) is contained within this single HTML file. The site uses Tailwind CSS for styling and includes JavaScript for navigation and form handling.

## How to Open and View the Website
1. Locate the `index.html` file in the project folder.

2. Double-click the `index.html` file to open it in your default web browser. Alternatively, you can right-click the file and select "Open with" to choose a specific browser.

## Regarding V1
- <b>Static Content:</b> This is a static site, meaning it's not yet connected to a database
- <b>Form Handling:</b> The "contact us" form is currently demo-only with a "thank you" message on submission. No data is sent or stored. The form will be connected to Supabase in V2.
- <b>Description:</b> The site includes placeholder text and images. Content will be updated in future versions.
